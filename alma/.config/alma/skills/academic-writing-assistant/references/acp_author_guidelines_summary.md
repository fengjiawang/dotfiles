# ACP Author Guidelines Summary

## Key Requirements for Atmospheric Science Papers

### 1. Manuscript Structure
- **Title**: Clear and concise, reflecting main findings
- **Abstract**: 150-250 words, structured (optional)
- **Keywords**: 3-6 relevant keywords
- **Introduction**: Context, literature review, research gap, objectives
- **Methods**: Detailed description of data, models, and analysis techniques
- **Results**: Clear presentation of findings with appropriate figures/tables
- **Discussion**: Interpretation of results, comparison with previous studies
- **Conclusions**: Summary of main findings and implications

### 2. Formatting Requirements
- **Language**: English only
- **Length**: Typically 10-20 pages (including references)
- **Figures/Tables**: High quality, numbered sequentially
- **Equations**: Numbered sequentially, properly formatted
- **References**: ACP citation format (see below)

### 3. Citation Format
ACP uses a numbered citation system:
- In-text: [1], [2,3], [4-6]
- Reference list: Numbered sequentially as they appear in text

### 4. Special Considerations for Atmospheric Science
- **Model descriptions**: Include version, configuration, parameterizations
- **Data sources**: Provide complete metadata and access information
- **Uncertainty analysis**: Include error bars and statistical significance
- **Code availability**: Encourage sharing of analysis code

### 5. Review Process
- Typically 2-3 reviewers
- Interactive discussion phase
- Technical corrections stage
- Final decision by editor
