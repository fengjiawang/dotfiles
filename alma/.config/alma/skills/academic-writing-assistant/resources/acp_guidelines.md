# ACP (Atmospheric Chemistry and Physics) Journal Guidelines

## Journal Overview
- **Full Name**: Atmospheric Chemistry and Physics
- **Publisher**: Copernicus Publications
- **Impact Factor**: ~7.0 (varies annually)
- **Scope**: Atmospheric chemistry, physics, and related Earth system sciences
- **Special Issues**: ACP Discussions (interactive peer review)

## Manuscript Types
1. **Research Articles**: Original research (typically 15-25 pages)
2. **Review Articles**: Comprehensive reviews of specific topics
3. **Brief Communications**: Short articles on timely topics
4. **Comments/Replies**: Discussion of previously published articles
5. **Special Issues**: Thematic collections

## Submission Requirements

### Length Guidelines
- **Abstract**: 250-300 words
- **Main text**: No strict limit, but typically 5000-8000 words
- **Figures**: 8-12 figures recommended
- **Tables**: As needed, but not excessive
- **References**: Comprehensive but relevant

### Structure Requirements
```
1. Title
2. Authors and affiliations
3. Abstract
4. Keywords (5-8)
5. Main text sections:
   - Introduction
   - Methods
   - Results
   - Discussion
   - Conclusions
6. Code and data availability
7. Author contributions
8. Competing interests
9. Acknowledgements
10. References
```

## Formatting Guidelines

### Text Formatting
- **Language**: English (British or American, but consistent)
- **Font**: Times New Roman, 12 pt
- **Line spacing**: 1.5 or double-spaced
- **Margins**: 2.5 cm on all sides
- **Page numbers**: Bottom center

### Equations
- Number consecutively: (1), (2), etc.
- Use MathType or LaTeX
- Define all variables when first used
- Use SI units

### References
- **Style**: Author-year (Copernicus format)
- **In-text**: (Author, Year) or Author (Year)
- **Multiple authors**: (Author1 et al., Year) for 3+ authors
- **List**: Alphabetical by first author's last name

### Figures and Tables
- **Placement**: At appropriate positions in text
- **Captions**: Below figures, above tables
- **Resolution**: Minimum 300 dpi
- **Formats**: TIFF, EPS, PDF preferred
- **Color**: CMYK for publication

## Special Requirements for Atmospheric Science

### Model Description
- Complete model specifications
- Parameterization schemes used
- Initial and boundary conditions
- Spin-up period description
- Validation against observations

### Data Analysis
- Statistical methods clearly described
- Uncertainty quantification
- Significance testing methods
- Correction for multiple testing if applicable

### Code and Data Availability
- **Mandatory**: Must provide access to code and data
- **Repository**: Zenodo, GitHub, or institutional repository
- **DOI**: Obtain DOI for datasets
- **License**: Specify usage license

## Review Process

### Interactive Discussion
1. **Initial submission**: Technical check
2. **Discussion phase**: Open peer review (8 weeks)
3. **Revised manuscript**: Address comments
4. **Final decision**: Editor's decision

### Response to Comments
- Address all referee comments
- Provide point-by-point responses
- Mark changes in revised manuscript
- Be professional and constructive

## Common Reasons for Rejection

### Technical Issues
- Insufficient model description
- Poor data quality or validation
- Inadequate statistical analysis
- Lack of uncertainty quantification

### Scientific Issues
- Incremental advance without novelty
- Poorly defined research question
- Inconsistent or contradictory results
- Overinterpretation of results

### Presentation Issues
- Poor English language quality
- Unclear figures or tables
- Incomplete references
- Missing data/code availability

## Tips for Success

### Before Submission
1. **Read recent ACP articles** in your field
2. **Check scope alignment** with journal
3. **Use ACP LaTeX template** (provided by Copernicus)
4. **Get language editing** if needed
5. **Prepare data/code repositories**

### During Writing
1. **Clear research question** in introduction
2. **Detailed methods** for reproducibility
3. **Logical results presentation**
4. **Thorough discussion** of mechanisms
5. **Specific conclusions** with implications

### After Submission
1. **Monitor discussion phase** actively
2. **Respond promptly** to comments
3. **Be open to criticism** and suggestions
4. **Submit revised manuscript** within deadline

## Special Considerations for Dust-Monsoon Research

### Required Elements
1. **Dust representation**: Emission, transport, removal processes
2. **Radiative transfer**: Dust optical properties, mixing state
3. **Model validation**: Against AERONET, MODIS, CALIPSO
4. **Mechanism analysis**: Process-based understanding
5. **Uncertainty assessment**: Model sensitivity, natural variability

### Recommended Analyses
1. **Radiative forcing**: Direct, semi-direct effects
2. **Circulation response**: Wind patterns, pressure systems
3. **Hydrological cycle**: Precipitation, evaporation, moisture transport
4. **Teleconnections**: ENSO, IOD, MJO interactions
5. **Future implications**: Climate change scenarios

### Common Pitfalls to Avoid
1. **Overlooking dust size distribution** effects
2. **Ignoring dust absorption properties**
3. **Not considering dust vertical distribution**
4. **Missing comparison with observations**
5. **Overstating conclusions** beyond evidence
